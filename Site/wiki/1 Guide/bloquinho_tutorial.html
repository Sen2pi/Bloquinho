Complete Bloquinho Features Tutorial

This comprehensive guide covers all Bloquinho features including the powerful editor, slash commands, formatting options, color customization, tables, diagrams, and more. Learn how to create professional documents with this Notion-like editor.

1. Editor Interface Overview

The Bloquinho editor provides a clean, professional interface similar to Notion. The main components include:

  • Toolbar: Formatting buttons and export options
  • Slash Menu: Access to all block types and commands
  • Editor Area: Endless page where you can add content
  • Preview Toggle: Switch between edit and preview modes
  • Auto-save: Your content is automatically saved as you type

Pro Tip: The editor supports both light and dark themes. You can switch themes in the settings menu.

3. Text Color and Background Customization

Text Color

Customize text colors to highlight important information or create visual hierarchy. Select any text and choose from a wide range of colors.

Background Color

Add background colors to text blocks for emphasis or to create visual sections within your documents.

The color configuration panel provides advanced options for both text and background colors, allowing you to create professional-looking documents with consistent styling.

4. Code Blocks and Syntax Highlighting

Bloquinho supports advanced code blocks with syntax highlighting for multiple programming languages:

  • JavaScript: Full ES6+ support with proper highlighting
  • Python: Complete Python syntax including decorators and type hints
  • Dart: Flutter and Dart-specific syntax
  • HTML/CSS: Web development languages
  • JSON: Data format with proper structure highlighting
  • Generic: Plain text with basic formatting

Pro Tip: You can specify the language after the opening backticks for automatic syntax highlighting: ```python

5. Mathematical Expressions and Matrices

Bloquinho supports comprehensive mathematical notation using LaTeX:

  • Basic Math: Fractions, powers, roots, and basic operations
  • Advanced Math: Integrals, summations, limits, derivatives
  • Matrices: Various matrix types (pmatrix, bmatrix, vmatrix)
  • Equations: Both inline and block equations
  • Greek Letters: Full Greek alphabet support

Example: Use $\begin{pmatrix} a & b \\ c & d \end{pmatrix}$ to create a 2x2 matrix.

6. Diagrams and Flowcharts

Create professional diagrams using Mermaid syntax:

  • Flowcharts: Process flows and decision trees
  • Sequence Diagrams: System interactions and API flows
  • Class Diagrams: Object-oriented design
  • ER Diagrams: Database relationships
  • Gantt Charts: Project timelines and schedules

Pro Tip: Diagrams are automatically rendered and can be exported as images for presentations.

7. Database and Table Features

Database Overview

The database feature provides Notion-like functionality for organizing and managing structured data.

Table Creation

Create custom tables with different column types and data validation.

Database features include:

  • Multiple Views: Table, Kanban, Calendar, and List views
  • Column Types: Text, Number, Date, Status, Select, and more
  • Filtering: Advanced filtering and sorting options
  • Relationships: Link tables and create relationships
  • Formulas: Calculated fields and mathematical operations

8. AI Integration

Bloquinho includes AI-powered features to enhance your productivity:

  • Content Generation: Generate text, lists, and structured content
  • Code Assistance: Get help with programming tasks
  • Documentation: Auto-generate documentation from code
  • Translation: Translate content between languages
  • Summarization: Create summaries of long documents

Pro Tip: Use the /ia command to quickly access AI features.

9. Export and Sharing

Bloquinho supports multiple export formats:

  • PDF: Professional PDF export with custom templates
  • Markdown: Clean markdown export for version control
  • HTML: Web-ready HTML with embedded styles
  • Plain Text: Simple text export for compatibility
  • Images: Export diagrams and charts as images

Custom PDF templates allow you to create professional documents with headers, footers, and custom styling for different use cases.

10. Formatting Toolbar

The formatting toolbar provides quick access to common formatting options:

  • Text Formatting: Bold, italic, underline, strikethrough
  • Alignment: Left, center, right, and justify alignment
  • Lists: Bullet points, numbered lists, and checkboxes
  • Links: Insert and edit links
  • Preview: Toggle between edit and preview modes

11. Advanced Features

Page Management

Create, organize, and link pages within your workspace. Use the page tree to navigate between documents and create hierarchical structures.

Version Control

Track changes and revert to previous versions of your documents. The system automatically saves versions as you work.

Collaboration

Share documents with team members and collaborate in real-time. Comments and annotations help with feedback and review processes.

Cloud Sync

All your documents are automatically synced across devices using Google Drive or OneDrive integration.

Search and Navigation

Powerful search functionality helps you find content quickly across all your documents and workspaces.

13. Best Practices

Document Organization

  • Use clear, descriptive titles for your documents
  • Organize content with proper heading hierarchy (H1, H2, H3)
  • Use tables for structured data and comparisons
  • Include diagrams for complex processes and relationships
  • Add code blocks with appropriate language tags

Formatting Tips

  • Use color sparingly to highlight important information
  • Maintain consistent formatting throughout your documents
  • Use callouts and quotes to emphasize key points
  • Include table of contents for long documents
  • Add links to related content and external resources

Collaboration

  • Use comments to provide feedback and suggestions
  • Create templates for recurring document types
  • Share documents with appropriate permissions
  • Use version control to track important changes
  • Export documents in appropriate formats for different audiences

FAQ

  • Is Bloquinho really free?
    Yes! Bloquinho is 100% free and open source (MIT License). No ads, no tracking, no paid features.
  • How do I install Bloquinho?
    Download the latest release for your platform (Linux, Windows, macOS) from GitHub. See the downloads section above.
  • Is my data private?
    Yes. All data is stored locally and encrypted. Cloud sync is optional and uses OAuth2.
  • Can I use Bloquinho offline?
    Absolutely. All features work offline. Cloud sync is optional.
  • How do I contribute?
    Visit the GitHub repository, open issues, submit pull requests, or join the community discussions.
  • How does the wiki work?
    Every HTML file in /wiki is a page. Add new files to expand the documentation. The menu updates automatically.
  • Where can I get help?
    Check the wiki, open an issue on GitHub, or contact the developer via LinkedIn or email.
